// Q1. What is a list in Python? What is another name for it?

Ans: A list is an ordered collection of items. Another name is sequence.

Q2. What are the 4 key characteristics of a list?
Ordered
Mutable (can be changed)
Allows duplicate values
Can store different data types

Q3. Difference between homogeneous and heterogeneous list?
Homogeneous: Same data type. Example: [1, 2, 3]
Heterogeneous: Different data types. Example: [1, "Python", 3.5]

Q4. Difference between basic and container data types? Name container types.
Basic: Stores a single value (int, float, str, bool).
Container: Stores multiple values.
Container types: List, Tuple, Set, Dictionary.

Q5. What is list indexing? Difference between positive and negative indexing?
Indexing: Accessing elements by position.
Positive: Starts from 0 (left to right).
Negative: Starts from -1 (right to left).

Q6. First and last index?
First element: 0
Last element (negative indexing): -1

Q7. What is list slicing? Syntax?
Ans: Slicing extracts a part of a list.
Syntax: list[start:end:step]

Q8. End index is exclusive—meaning?
Ans: The end index is not included.
Example: a = [10,20,30,40]
a[1:3] → [20,30]

Q9. What does list[:] return?
Ans: It returns a copy of the entire list.

Q10. Mutable vs Immutable. Is list mutable?
Mutable: Can be modified.
Immutable: Cannot be modified.
List is mutable.

Q11. Are duplicates allowed?
 Yes.
Example: [1, 2, 2, 3]

Q12. Loop through a list using for loop.
  for item in list:
    print(item)

Q13. What is enumerate()? When to use it?
 enumerate() gives both index and value. Use it when you need the index.

Q14. Difference between for and while loop?
for: Simpler, directly iterates over elements.
while: Uses an index and condition.
Q15. What does len() return?

Ans:

len([]) → 0
len([1,2,3,4,5]) → 5
